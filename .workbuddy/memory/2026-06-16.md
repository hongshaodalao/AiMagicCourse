## 音效生成
- 使用 Python (numpy + scipy) 程序化生成5段约5秒的音效 WAV 文件
- 雨声 (rain.wav): 棕噪声 + 低通滤波 + 滴答声
- 猫叫 (cat_meow.wav): 频率扫描 (700→350→550Hz) + 谐波 + 振幅包络
- 火车 (train.wav): 低频节奏 chugging (4Hz) + 汽笛声
- 海浪 (ocean_waves.wav): 调制噪声 + 低频膨胀包络 + 浪花高频
- 夜晚森林 (night_forest.wav): 蟋蟀鸣叫 (4-5kHz脉冲调制) + 风声 + 猫头鹰叫声
- 输出目录: sounds/ 下，44100Hz 16-bit 单声道
- 生成脚本: generate_sounds.py
